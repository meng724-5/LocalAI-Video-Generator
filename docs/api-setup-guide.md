# API申请教程

## 一、豆包开放平台 API Key 申请

### 1. 注册火山引擎账号
1. 访问 [火山引擎官网](https://www.volcengine.com/)
2. 点击右上角"注册/登录"
3. 使用手机号注册并完成实名认证

### 2. 开通方舟大模型服务
1. 登录后进入 [火山引擎控制台](https://console.volcengine.com/)
2. 在顶部搜索框输入"方舟"或"大模型"
3. 点击进入"方舟大模型"服务页面
4. 点击"立即开通"按钮

### 3. 创建 API Key
1. 进入方舟大模型控制台
2. 左侧菜单选择"API Key管理"
3. 点击"创建API Key"
4. 输入名称（如：LocalAI-Video-Generator）
5. 复制生成的API Key（注意：只显示一次，请妥善保存）

### 4. 充值或确认免费额度
1. 进入"费用中心"
2. 查看账户余额和免费额度
3. 新用户通常有一定量的免费token额度
4. 如需充值，点击"充值"按钮，最低充值金额通常为10元

## 二、火山引擎语音合成 API 申请

### 1. 开通语音合成服务
1. 在火山引擎控制台顶部搜索"语音合成"
2. 点击进入"语音技术"服务页面
3. 找到"长文本语音合成"或"语音合成"
4. 点击"立即开通"

### 2. 创建应用获取 App ID
1. 进入语音合成控制台
2. 左侧菜单选择"应用管理"
3. 点击"创建应用"
4. 输入应用名称（如：LocalAI-Video-TTS）
5. 选择服务类型：语音合成
6. 创建成功后，复制"App ID"

### 3. 获取 API Key
1. 在应用详情页面，找到"API Key"或"密钥管理"
2. 点击"创建密钥"
3. 复制生成的 API Key

### 4. 确认服务权限
1. 进入"权限管理"或"服务开通"页面
2. 确保"长文本语音合成"服务已开通
3. 查看免费额度和计费说明

---

## 三、密钥填写位置

1. 打开 LocalAI Video Generator 软件
2. 点击顶部导航栏的"API设置"
3. 在对应输入框填入：
   - 豆包开放平台 API Key
   - 火山引擎 App ID
   - 火山引擎 API Key
4. 点击"保存配置"

## 四、常见问题

**Q: 两个API可以用同一个火山引擎账号吗？**
A: 可以。豆包和火山语音合成都是字节火山引擎旗下的服务，使用同一个账号即可。

**Q: 免费额度用完了怎么办？**
A: 需要充值后才能继续使用。豆包大模型按token计费，语音合成按字符数计费，费用通常较低。

**Q: API Key泄露了怎么办？**
A: 立即到控制台删除该API Key并重新创建。软件中的密钥本地加密存储，请勿分享配置文件。

**Q: 为什么提示API调用失败？**
A: 请检查：
1. API Key是否填写正确（无多余空格）
2. 账户是否有余额或免费额度
3. 对应服务是否已开通
4. 网络连接是否正常
